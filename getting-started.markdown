# Getting Started
To begin, we need to install the software that we are going to be using throughout the module.

## Installing Chrome
I assume most of you are already using Chrome, or have Chrome installed on your machine already. Any modern web browser will be fine for the module, but Chrome gives you very good development tools which will be helpful for debugging your applications.

[Download Chrome](https://www.google.com/intl/en_uk/chrome/dr/download/)

## Installing NodeJS
Node is a JavaScript runtime environment that allows us to run JS applications outside of the browser. This enables us to create back-end applications in JS (as well as front end applications using frameworks).

[Download NodeJS](https://nodejs.org/en/download)
